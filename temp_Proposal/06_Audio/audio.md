# ICEBREAKER — 오디오 디렉션

> **상태: LOCKED · 2026-07-19 · 기준 문서: `new/ICEBREAKER_SPEC_v2.md` v2.6**  
> 전담 사운드 인력 없음 · 라이선스가 확인된 소스 + 편집·레이어·피치 변조로 제작

## 1. 한 문장 오디오 정의

**차갑고 깨끗한 얼음 파열음 위에 낮은 선박 기관음과 따뜻한 항구 신호음이 얹히는 소리.**

오디오는 세계관을 설명하지 않는다. 얼음의 단계, 선박의 존재, 항구에 가까워졌다는 사실을 소리로 보강한다.

## 2. 기본 원칙

- 업무 불간섭을 위해 최초 실행 시 음소거
- 유저가 켠 설정은 로컬 저장
- 음성 대사·내레이션·캐릭터 보이스 없음
- 공포용 저음 드론과 전쟁용 포성 배제
- 얼음 파괴음은 짧고 선명하게, 선박 소리는 낮고 지속적으로 구성
- **매 파괴마다 재화음을 따로 재생하지 않는다.** 동전이 쏟아지는 소리는 사용하지 않는다.
- **결정빙과 큰 연쇄에서만 짧은 강조음을 추가한다.**
- 같은 소리 반복을 줄이기 위해 피치·볼륨·시작 위치를 소폭 랜덤화
- 연쇄 파괴 때 개별 소리를 무제한 겹치지 않음
- 일정이 부족하면 배경음악을 먼저 제외한다.

### 감정 대비

| 영역 | 소리 | 감정 |
|---|---|---|
| 얼음바다 | 바람·얼음 마찰·얇은 금속성 고음 | 차가움·거리감 |
| 쇄빙선 | 낮은 디젤 기관·선체 진동 | 안정감·무게감 |
| 파쇄 | 짧은 충격·유리성 파열·파편 | 시원함·성취감 |
| 항구 | 뱃고동·따뜻한 화음 | 도착·안도감 |

## 3. 얼음 단계별 소리

하나의 기본 타격·파괴 소스를 레이어와 피치로 변형한다.

| 단계 | 피치 | 인상 |
|---|---:|---|
| T1 백빙 | 높게 | 높고 얇게, 가볍고 잘 깨짐 |
| T2 청빙 | 기준 | 단단하고 선명함 |
| T3 심빙 | 낮게 | 낮고 무겁고 깊음 |

- 일반 타격 피치 ±3%, 볼륨 ±1.5dB 랜덤 변조
- 좌우 패닝: 얼음 화면 위치에 따라 최대 ±25%
- 치명타 고음과 특수빙 고유음은 피치 랜덤 범위를 절반으로 줄여 인식성 유지

| 카운트다운 | 카운트다운 소리 + 낮은 엔진음 |

| 작업 종료 직전 | 타격·음악 0.1초 정지, 기관음은 아주 작게 유지 |
| 작업 종료·정산 | 작업 종료음 → 정산 금액이 나타날 때 짧은 확인음 1회 |
| 목적지 도착 | 정산 뒤 도착 연출과 함께 따뜻한 도착 화음 |
| 항해 중 | 기본 무음, 유저가 환경음을 켰을 때만 낮은 기관·바람 |

## 6. 연쇄 파괴 믹싱

- 동시 파괴음 보이스 최대 8개
- 0.03초 안에 같은 파괴음이 여러 번 발생하면 가장 큰 1개를 중심으로 묶음
- 9개째부터는 개별 파괴음 대신 연쇄 러시 레이어의 볼륨만 상승
- 연쇄 강조음은 최대 5단까지만 피치 상승, 이후 같은 최고 피치 유지
- 큰 연쇄 발생 시 환경음을 0.4초 동안 4dB 낮춤

## 7. 오디오 믹서 구조

```text
Master
├─ Music
├─ SFX_Gameplay
├─ SFX_Ship
├─ Ambience
└─ UI
```

| 그룹 | 기준 |
|---|---:|
| Master Peak | -1 dBFS 이하 |
| SFX_Gameplay | 기준 0 dB |
| SFX_Ship | Gameplay 대비 -4 dB |
| UI | Gameplay 대비 -8 dB |
| Ambience | Gameplay 대비 -12 dB |
| Music | Gameplay 대비 -10 dB |

절대 수치보다 파쇄음이 가장 먼저 들리고 기관음과 음악이 뒤에 남는 관계를 우선한다.

## 8. BGM

- 쇄빙 작업용 무봉제 루프 1곡 (선택)
- 파쇄 효과음을 가리지 않도록 1~4kHz 대역을 비움
- 항해 중은 기본 무음
- **일정 부족 시 배경음악을 가장 먼저 컷한다.** 음악이 없어도 게임 정체성이 성립해야 한다.

## 9. 파일 기준

### 원본

- WAV · 48kHz · 24bit 권장 · 피크 -1 dBFS 이하
- 루프 파일은 시작·끝 제로 크로싱 확인

### Unity 임포트

- 짧은 UI·파쇄음: Decompress On Load
- 1초 이상 환경음·음악: Vorbis 압축
- 스테레오가 필요 없는 충격음은 Mono
- 모든 오디오는 2D로 사용하며 3D 거리 감쇠는 사용하지 않음

## 10. 라이선스 관리

- CC0, 자체 제작, 구매한 상업 이용 가능 소스를 우선한다.
- 출처가 불명확한 영상·게임·유튜브 추출 음원 사용 금지
- 라이선스와 출처 URL을 `Audio/source_licenses.csv`에 기록
- CC-BY 사용 시 게임 크레딧과 제출 페이지에 저작자 표기

기록 필드:

```text
file_name, source_name, source_url, author, license, modified, credit_text
```

## 11. 오디오 승인 기준

- T1·T2·T3를 화면을 보지 않고도 대략 구분할 수 있음
- 20개 연쇄 파괴에서도 클리핑과 불쾌한 고음이 없음
- 파쇄음이 기관·바람·음악보다 먼저 들림
- 매 파괴마다 재화음이 나지 않고, 결정빙·큰 연쇄에서만 강조음이 남
- 소리 기본 꺼짐과 설정 저장이 동작함
- 정산 화면에서 확인음이 1회만 재생됨
- 축소 게임 바 상태에서 자동으로 소리가 재생되지 않음
