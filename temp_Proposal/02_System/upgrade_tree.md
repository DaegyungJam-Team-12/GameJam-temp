# 선박 정비 — 영구 강화 구조

모든 항목은 정비 자금으로 구매하고 쇄빙 작업이 끝나도 유지된다. **카운트다운과 60초 쇄빙 작업 중에는 구매할 수 없으며**, 항해 중·유빙 구간 도착·쇄빙 준비·항로 갱신 상태에서는 구매 가능하다.

## 구조 요약

- **공통 진행:** 운항 보수 효율 → 청빙 대응 → 심빙 대응 순서로 모든 플레이어가 진행한다.
- **정비 계통:** 직접 파쇄 / 보조 파쇄 / 연쇄 파괴를 원하는 순서로 섞어 구매할 수 있다. 상호 배타적 선택이 아니므로 P0에서는 `빌드`라고 부르지 않는다.
- **상시 자동 공격 없음:** 보조 장비는 플레이어의 클릭·파괴·연쇄가 일어났을 때만 충전·발동한다.
- **60초 불변:** 시간 연장 정비 항목은 만들지 않는다.

## 공통 루트 — 기본 파쇄 장비

| 비용 | 효과 |
|---:|---|
| 10 | 클릭 피해 ×2 · 구매 시 공통 진행과 정비 계통 3개 개방 |

## 공통 진행 — 빙질 대응

정비 계통 선택과 별개로 더 단단한 해역으로 나아가기 위한 필수 진행선이다.

| 항목 | 랭크·비용 | 효과 |
|---|---|---|
| P1 운항 효율 | 3랭크 · 50 / 300 / 1,800 | 랭크당 운항 보수 +25% |
| P2 청빙 대응 | 500 | 섬마을 도착 후 구매 가능 · T2 해금 · 구간 단계 2 진입 후 첫 10회 스폰 안에 청빙 확정 등장 |
| P3 심빙 대응 | 8,000 | 등대항 도착 후 구매 가능 · T3 해금 · 구간 단계 3 진입 후 첫 10회 스폰 안에 심빙 확정 등장 |

## A. 직접 파쇄 — 한 목표를 강하게

| 항목 | 랭크·비용 | 효과 |
|---|---|---|
| A1 주 파쇄기 출력 | 3랭크 · 25 / 150 / 900 | 랭크당 클릭 피해 ×1.6 |
| A2 고속 장전 | 2랭크 · 80 / 600 | 홀드 초당 5→7→10회 |
| A3 충격 전이 | 8,000 | 초과 피해의 50%가 가까운 얼음 2개로 전이 |

## B. 보조 파쇄 — 직접 조작에 추가 타격

| 항목 | 비용 | 효과 |
|---|---:|---|
| B1 보조포 충전 | 100 | 직접 클릭 10회마다 가까운 다른 얼음 1개를 클릭 피해 100%로 추가 타격 |
| B2 동시 사격 | 600 | 직접 클릭할 때마다 가까운 다른 얼음 1개에 클릭 피해 25% 적용 |
| B3 보조 파쇄기 | 2,500 | 5연쇄 달성 시 3초 동안 초당 1회, 얼음 1개를 클릭 피해 100%로 타격 · 발동 중 재발동 없음 |

보조 파쇄 피해는 직접 클릭으로 취급하지 않아 스스로 다시 충전하지 않는다. 플레이어가 손을 떼면 신규 충전과 신규 발동은 멈춘다. 이미 발동한 B3의 3초 사격은 남은 지속시간을 마치고 종료한다.

## C. 연쇄 파괴 — 충격을 주변으로 전파

| 항목 | 비용 | 효과 |
|---|---:|---|
| C1 파편 비산 | 200 | 모든 파괴가 최근접 얼음 2개에 최대 HP 10% 피해 |
| C2 충격 증폭 | 800 | P0 특수빙 2종의 효과 피해 +50% |
| C3 빙판 붕괴 | 12,000 | 매 20번째 파괴마다 화면의 모든 얼음에 최대 HP 25% 피해 |

## P0 효과 계산 규칙

구현자가 계통별 효과를 다르게 해석하지 않도록 아래 순서를 고정한다.

```text
clickDamage = baseDamage × 기본 파쇄 장비 배율 × 1.6^(A1 랭크)
directDamage = clickDamage × (크리티컬이면 5, 아니면 1)
운항 효율 배율 = 1 + 0.25 × P1 랭크
```

- A1은 랭크마다 이전 값에 ×1.6을 곱한다.
- B1·B2·B3는 `clickDamage`를 기준으로 계산하며 자체 크리티컬이 발생하지 않는다.
- B2의 25% 피해는 소수값을 유지하고 HP 판정까지 반올림하지 않는다.
- 특수빙과 C 계통 피해는 대상 최대 HP 비율로 계산한다. C2는 비율 피해에 ×1.5를 적용하되 100%를 넘지 않는다.
- 하위 티어 즉시 파괴처럼 명시된 규칙은 비율 피해 계산보다 우선한다.
- 운항 보수는 작업 중 소수값으로 누적하고 정산 시 최종 합계를 정수로 반올림한다.

## P1 확장 후보

급소 포착 · 연쇄 운항 보수 · 시작 구간 단계 상향 · T4/T5 빙질 대응 · 특수빙 출현 확률 증가. **P1에서도 상시 자동 공격은 기본 계획으로 두지 않는다.**

## 설계 규칙

- 빙질 대응은 정비 계통 선택과 관계없이 모든 플레이어가 거치는 진행 축이다.
- 보조 파쇄는 직접 조작의 보상이며 방치만으로 쇄빙 작업을 완료하지 않는다.
- 선박 정비는 항구 도착 여부와 관계없이 60초 작업 사이에 가능하다.
- 완성 그림: 강한 직접 타격 → 보조 장비 추가 타격 → 파편 비산 → 특수빙 연쇄 → 빙판 붕괴가 연달아 발생한다.
