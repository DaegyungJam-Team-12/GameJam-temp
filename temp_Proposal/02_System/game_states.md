# 게임 상태 흐름 — 7단계 + 마지막 5초 이벤트

> 기준 문서: `01_Concept/concept.md` v3.2  
> 60초 쇄빙 작업은 성공·실패 판정이 아니라 고정 길이의 성과 측정 구간이다.

| # | 상태 | 내용 |
|---|---|---|
| 1 | 항해 중 | taskbar 도킹 상태. 확보한 수로를 따라 다음 유빙 구간으로 이동한다. 선박 정비·기록·항로 확인 가능. 앱 종료 중에는 타임스탬프 차분으로 대기 시간만 진행한다. |
| 2 | 유빙 구간 도착 | 다음 쇄빙 작업 시작 가능 상태. **자동 시작 없음.** 스트립에 다음 얼음 실루엣과 준비 완료 표시가 나타난다. |
| 3 | 쇄빙 준비 | 유저가 스트립을 눌러 확장한다. 목적지·화물·항로 진행·보유 정비 자금·최고 기록을 확인한다. 시작 전 시간제한 없음. |
| 4 | 카운트다운 | 3·2·1. 3초 동안 유빙 구간과 얼음이 드러나며 마지막 숫자와 동시에 입력을 활성화한다. |
| 5 | 쇄빙 작업 | 정확히 60초. 얼음 파괴·재스폰·전진 점수·운항 보수 누적. 작업 중 **선박 정비 구매 불가**, **접기 비활성**, 일시정지 없음. 남은 시간이 5초가 되면 입력과 파쇄 규칙은 유지한 채 경고 이벤트만 발생한다. |
| 6 | 운항 정산 | 0.3초 정지 → 남은 얼음 이동 정지·동결 → 이번 운항 보수를 정비 자금에 반영하고 전진 점수를 항로 진행에 더한다. 다음 구간 대기 타이머는 이 순간 시작한다. |
| 7 | 항로 갱신·복귀 | 결과와 갱신된 항로 위치를 표시하고 선박 정비를 허용한다. 항로 목표 도달 시 1~2초 공통 항구 도착 연출 후 다음 항로를 연다. 북쪽 항로도 목표값 도달로 완료하며 전용 엔딩은 필수 상태에 포함하지 않는다. 접으면 1로 순환한다. |

## 마지막 5초 이벤트

- 별도 게임 상태가 아니라 `쇄빙 작업` 안에서 한 번 발생하는 프레젠테이션 이벤트다.
- 클릭·홀드·파괴·재스폰·보수 누적은 0초까지 그대로 유지한다.
- 타이머 크기·경고색과 화면 가장자리 성에를 강조하고 낮은 펄스 레이어만 추가한다.
- BGM 속도와 타격음 피치는 바꾸지 않는다.
- 0초 직전 0.1초 동안 Gameplay·Music 그룹만 멈추고 기관음은 매우 작게 유지한다.

## 첫 실행과 반복

- 최초 실행은 대기 없이 **2. 유빙 구간 도착**에서 시작한다.
- 6단계에서 대기 타이머가 즉시 시작되므로 결과·정비 화면을 보는 시간도 다음 구간 준비에 포함된다.
- 대기 시간이 끝나도 2단계에 머무르며 유저 입력 전까지 쇄빙 작업을 시작하지 않는다.
- 항해 중에는 자동 파쇄·자동 운항 보수·자동 항로 진행이 없다.

## 다음 유빙 구간까지 시간

| 빌드 | 시간 |
|---|---|
| 최초 실행 | 즉시 플레이 가능 |
| 표준 데이터 프로필 초기값 | **30초** |
| 발표·시연 데이터 프로필 초기값 | 10초 |
| 장기 대기 | P0 플레이테스트 뒤 재검토 |

30초는 정산 확인과 선박 정비를 하는 동안 지나가는 시작값이다. D5~D10 플레이테스트에서 대기 이탈률과 정비 시간을 보고 조정하며 D11에 잠근다. 시연 빌드는 별도 10초 데이터 프로필로 두 번째 플레이를 보장한다. 공개 설정 화면에는 프로필 전환을 노출하지 않는다.

## 항로 완료 분기

운항 정산에서 다음 순서로 판정한다.

1. 이번 쇄빙 작업의 전진 점수를 현재 항로 진행에 1:1로 더한다.
2. 현재 항로 진행이 목표값 이상이면 목표값으로 고정한다. 초과 점수는 다음 항로로 넘기지 않는다.
3. 일반 항로는 목표값 도달 시 항구 도착 처리한다.
4. 최종 항로도 목표값 도달 시 북쪽 기지 도착 처리한다. `T3_FIRST_BREAK`는 최고 빙질·성장 기록으로만 저장한다.
5. 얼음에서 획득한 운항 보수는 항로 진행 초과 여부와 관계없이 전액 지급한다.

북쪽 기지 전용 엔딩 화면은 위 상태 흐름에 포함하지 않는다. 표준 RC가 안정되고 시간이 남을 때만 7단계 뒤에 붙는 선택 연출로 구현한다.

## 운항 정산 표시

- 이번 운항 보수
- 파괴한 얼음 수
- 최고 빙질
- 최대 연쇄
- 획득 전진 점수
- 현재 항로 진행
- 직전 쇄빙 작업 대비 운항 보수 증가율

P1에서는 직전 기록 그래프를 추가할 수 있다.
