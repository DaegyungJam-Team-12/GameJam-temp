# P0 (게임잼) 범위 잠금

> **상태: LOCKED · 2026-07-19 · 기준 문서: `new/ICEBREAKER_SPEC_v2.md` v2.6 §10**  
> 목표는 새 기능을 많이 보여주는 것이 아니라, **60초 쇄빙 → 자금 정산 → 30초 항해·정비 → 다음 쇄빙 → 목적지 도착**이 하나의 완결된 게임으로 이어지는 것이다.

## 1. P0 성공 기준

- 표준 데이터에서 신규 플레이어가 목표 30~60분 안에 북쪽 기지에 도착한다. 시연 데이터로는 10분 이내에 전체 흐름을 보여줄 수 있다.
- 첫 5분 안에 `얼음 파괴 → 즉시 정비 자금 → 선박 정비 → 더 큰 파괴`의 성장 차이를 이해한다.
- 대사나 프롤로그 없이도 `화물을 실은 배가 얼음을 깨고 항구로 간다`는 상황이 보인다.
- 다른 PC에서 처음부터 목적지 도달까지 플레이 가능한 Windows 빌드가 나온다.
- 저장 후 재실행했을 때 자금·목적지 진행·구매·첫 파괴 여부가 유지된다.

## 2. 수요일(7/22)까지 기능 완성 — P0

- 800×72 축소 창과 960×540 확장 창
- 60초 쇄빙 작업과 30초 항해
- 클릭·누르고 있기·치명타
- 최대 20개 얼음과 재생성
- T1~T3, 결정빙, 균열빙의 기본 규칙
- 기본 보조 파쇄와 연쇄 파괴
- 즉시 정비 자금과 목적지 진행
- 정비·운항 현황·쇄빙 시작·설정을 제공하는 800×72 축소 런처
- 매 작업 종료 후 자금 정산 화면
- 선박 정비 13개 구매·적용
- 세 목적지 데이터와 공통 항구 도착 처리
- 로컬 저장, 표준·시연 데이터
- 한국어와 Windows 실행 파일
- 플레이 판독에 필요한 P0 아트

| 순서 | 목적지 | 화물 | 표준 목표 |
|---:|---|---|---:|
| 1 | 섬마을 | 식료품·우편 | 120 |
| 2 | 등대항 | 발전기 연료·의약품 | 600 |
| 3 | 북쪽 기지 | 기계 부품·우편 | 2,400 |

필수 완료 범위는 세 목적지 도달까지다. 별도 엔딩은 제출 후보가 안정된 뒤에만 검토한다.

## 3. 목·금(7/23~24) 대면 보완

- 타격감, 화면 흔들림, 연쇄·보조 파쇄 피드백 개선
- 정비 트리와 축소 런처의 가독성 보완
- 표준·시연 밸런스 조정
- 최종 아트·오디오 교체와 누락 수정
- 목적지 도착 연출 보완
- 다른 PC 실행, 저장, 해상도, 제출 빌드 검증

새 핵심 시스템보다 밸런스·연출·아트·오디오·제출 안정성을 보완한다.

- 장문 대화와 물류 시뮬레이션

- 동전·부품 드롭과 수동 획득
- 정산 화면에서 정비 자금 재지급
- T4·T5, 추가 특수빙, 추가 목적지·분기 (P1)
- macOS·Steam·온라인·분석 SDK

- 목요일(7/23) 전체 플레이를 기준으로 최종 수치를 확정한다.
