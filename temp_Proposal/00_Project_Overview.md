# ICEBREAKER — 프로젝트 개요

> 얼어붙은 바다의 마지막 쇄빙 보급선  
> 게임잼 「납량(서늘함을 들인다)」 출품작 · 2026.07  
> **최상위 기준 문서: `new/ICEBREAKER_SPEC_v2.md` (통합 기획서 v2.6)** — 이 폴더의 문서와 해석이 충돌하면 통합 기획서를 우선한다.

## 한 줄 소개

**작업 표시줄 위의 쇄빙선을 열어 60초 동안 얼음을 부수고, 정비 자금으로 배를 강화해 세 목적지까지 가는 PC 게임.**

## 유저 카피

**얼어붙은 바다를 뚫고, 끊어진 항로를 잇는다.**

## 플레이어가 하는 일

1. 축소 게임 바에서 정비·운항 현황을 확인하고 쇄빙 시작을 누른다.
2. 60초 동안 최대 20개의 얼음을 클릭·누르고 있기로 부순다.
3. 얼음을 파괴한 순간 정비 자금과 목적지 진행을 얻는다.
4. 작업 종료 뒤 자금 정산 화면에서 이번 작업 성과를 확인한다.
5. 30초 항해 동안 선박 정비 13개로 배를 강화하며 세 목적지에 보급품을 전달한다.

## 핵심 스펙

| 항목 | P0 확정 |
|---|---|
| 장르 | 파쇄·운항형 증분 × 60초 액션 × taskbar 컴패니언 |
| 코어 판 | 쇄빙 작업 60초 · 자유 위치 · 최대 20개 얼음 |
| 성장 | T1~T3 얼음 단계 + 선박 정비 13개 (공통·직접·보조·연쇄) |
| 재화 | 정비 자금 1종 · 얼음 파괴 즉시 지급 |
| 장기 목표 | 섬마을 → 등대항 → 북쪽 기지 세 목적지 도달 (120 / 600 / 2,400) |
| 판 사이 | 30초 항해 · 매 작업 뒤 자금 정산 화면 · 자동 파쇄·방치 수익 없음 |
| 창 | 480×56 축소 게임 바(메인 진입점) / 720×480 확장 |
| 플랫폼 | Windows · Unity 6 |
| 언어 | 한국어 (영어는 일정 여유 시) |
| 목표 완주 | 표준 30~60분 · 시연 데이터로 10분 이내 |

## 성장의 정의

숫자가 커지는 것보다 **한 번의 입력으로 화면에서 함께 깨지는 얼음이 늘어나는 것**을 먼저 보여준다.

```text
한 얼음을 더 빠르게 파쇄
→ 보조 파쇄기가 다른 얼음을 추가 타격
→ 파편과 연쇄가 주변으로 전파
→ 더 단단한 얼음 단계를 열고 이전 얼음을 화면째 정리
```

상위 얼음은 HP 증가보다 정비 자금 증가가 크다. 새 얼음을 안정적으로 부술 수 있게 되는 순간부터 자금 효율이 높아지는 것이 성장 보상이다.

## 이야기 전달 원칙

- 프롤로그·대화·컷신으로 세계를 설명하지 않는다.
- 작업 표시줄 위의 작은 선박 게임 바, 멀리 보이는 목적지 불빛, 적재된 화물 이름, 앞으로 나아가는 배로 목적을 전달한다.
- 화물은 P0에서 시각 정보이며 인벤토리나 능력치가 아니다.
- 목적지 도착은 항구 이름 표식과 불빛 변화의 1~2초 공통 연출이다.

## P0 / P1 경계

### P0 — 게임잼에 반드시 완결 (7/22 수요일 기능 완성)

- T1~T3, 결정빙·균열빙
- 세 목적지 데이터와 공통 항구 도착 처리 — 북쪽 기지 도달이 P0 완료 조건
- 선박 정비 13개, 로컬 저장, 표준·시연 데이터, 한국어
- 480×56 축소 런처와 720×480 쇄빙·정비·운항 현황·정산 화면
- 캐주얼 손그림풍 게임 화면과 선장 반신 원화 1장

별도 엔딩은 제출 후보가 안정된 뒤에만 검토한다.

### P1 — 게임잼 이후 검토

- T4·T5와 추가 특수빙, 추가 목적지와 분기 경로
- 의뢰 선택과 화물 보정, 통계와 기록 비교 확장
- 영어를 포함한 추가 언어, macOS·Steam 출시 준비

자동 공격, 방치 수익, 거래 경제는 P0·P1 모두 포함하지 않는다.
